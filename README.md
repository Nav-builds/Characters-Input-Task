

# 📝 Real-Time Character Counter

A simple and interactive **real-time character counter** built using **HTML, CSS, and JavaScript**. This project dynamically updates the number of characters typed into a text area, providing instant feedback to users.

## 🚀 Features

* 🔢 Real-time character counting as you type
* ⚡ Instant UI updates without page reload
* 🎯 Displays total characters and remaining limit (if applicable)
* 💡 Clean and responsive user interface
* 🧠 Beginner-friendly JavaScript logic

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (Vanilla)** – Functionality and real-time updates

## 📂 Project Structure

```
├── index.html      # Main HTML file
├── style.css       # Styling file
├── script.js       # JavaScript logic
```

## 📸 How It Works

1. User types into the text area
2. JavaScript listens for input events
3. Character count updates instantly
4. UI reflects current character usage
